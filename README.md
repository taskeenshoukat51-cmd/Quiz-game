# Quiz-game
Interactive quiz game with score tracking and file handling built using advanced C++ features.
## Tech Used
- C++
- File Handling

## Overview
An interactive console-based quiz game built in C++ using 
Object Oriented Programming concepts. The player answers 
10 multiple choice questions from different topics and 
gets a final score with pass or fail result.

---

## How It Works
1. Player enters their Name, Age and Date of Birth
2. Player is asked if they are ready to start
3. 10 multiple choice questions are displayed one by one
4. Player selects answer by entering option number (1-4)
5. Correct or Wrong feedback is shown after each answer
6. Final total score is displayed at the end
7. Pass (70+) or Fail result is announced

---

## Features
- Player details input (Name, Age, Date of Birth)
- 10 Multiple Choice Questions
- Each question carries 10 marks
- Total score out of 100
- Instant feedback after every answer
- Pass/Fail result at the end
- Colored console interface

|---|---|
| 70 - 100 | ✅ Pass — Congratulations! |
| 0 - 69 | ❌ Fail — Better luck next time |

## OOP Concepts Used
- **Class:** Question class with private attributes
- **Objects:** 10 question objects (q1 to q10)
- **Encapsulation:** Private data members
- **Setter Function:** setValues() to set question data
- **Member Function:** askQuestion() to display and check answers
